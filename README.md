# Mi Gestor de Negocio

Sistema de gestion para pequenos negocios (ventas, inventario, reportes) que funciona sin internet ni instalacion. Solo abre el archivo en tu navegador.

## Caracteristicas

- Panel de control: ventas del dia, total productos, alertas de stock bajo
- Gestion de productos: agregar, editar, eliminar y buscar productos
- Registro de ventas: vender con descuento automatico de stock
- Anular ventas: devuelve el stock al producto
- Cambio de moneda: personaliza el simbolo ($, EUR, S/, etc.)
- Reportes CSV: exporta productos, ventas totales o ventas del dia
- Persistencia local: los datos se guardan automaticamente

## Como usarlo

### Opcion 1: Descargar y abrir
1. Descarga el archivo `index.html`
2. Haz doble clic o abrelo con tu navegador
3. Empieza a gestionar tu negocio
